# Experiment Proposal

## Title
Welcome message impact on Day 1 activation.

## Problem
Activation rate is 42% - 58% of new users never send a first message.

## Hypothesis
If we show new users a welcome message with 3 suggested prompts, then day 1 activation rate will increase by at least 10% relative (42% - 46%) because users don't know what to ask first.

## Change
Show a welcome message with 3 suggested use cases on the empty input screen for new users only.

## Primary Metric
day 1 activation rate % of new users who send their first message.

## Secondary Metrics
- number of messages sent in first session
- day 7 retention

## Success Criteria
activation rate increases by at least 5 percentage points, p < 0.05.

## Sample Size
4386 total users (2192 per group), based on 42% baseline, 10% relative MDE, 80% power, p < 0.05.

## Test Duration
14 days

## Risks
- welcome message reduces perceived product sophistication, attracting low-intent users who activate but don't retain.
- What if activation goes up but message quality goes down?
